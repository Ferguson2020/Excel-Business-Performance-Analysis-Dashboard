Excel Business Performance Analysis Dashboard
🚀 Project Overview

This project is a data-driven business analysis built entirely in Microsoft Excel. It focuses on transforming raw business data into meaningful insights using data cleaning techniques, formulas, pivot tables, and interactive dashboards.

The goal is to evaluate company performance, identify trends, and support data-informed decision-making.

🎯 Objectives

Analyze revenue and cost performance

Track key performance indicators (KPIs)

Identify growth trends and patterns

Highlight operational inefficiencies

Present insights using a dynamic dashboard

🛠️ Tools & Excel Features Used

Pivot Tables & Pivot Charts

Excel Formulas (IF, VLOOKUP/XLOOKUP, SUMIFS, etc.)

Data Cleaning & Formatting

Conditional Formatting

KPI Summary Cards

Interactive Filters (Slicers)
